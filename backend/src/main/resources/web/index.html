What is Scala.js?

Basically, Scala-to-JS compiler

Why?

  • Share code between client and server
  • Strong typing, including for JavaScript libraries
  • Tooling: IDE support, Scala std-lib, ...
  • Fast compilation cycle (compared to other JVM-JS such as GWT)
  • Great community, good documentation
according to this thread

  • Google: Dart & AtScript; Closure
  • Facebook: Flow

How?

  • compiles scala AST to .class and .sjsir
  • does not compile .class to .sjsir
  • compiles .sjsir to JS, optionally optimizing with advanced Closure
  • produces single JS file

Downsides

  • debugging (same as *-to-js)
  • has source-maps though

  • NIH: difficult to draw from JS ecosystem (npm, bower support?)
